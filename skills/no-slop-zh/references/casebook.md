## 社区案例

### 1. 2026-02-05: “稳稳接住你”成为独立吐槽主题

LINUX DO 主题《我就在这里，稳稳地接住你》中，用户直接把这句话当成可识别口癖，评价是“好窒息”，并指出很多模型都有类似的八股句式。

来源：[我就在这里，稳稳地接住你](https://linux.do/t/topic/1568563)

### 2. 2026-03-06: “如果你愿意”被单独点名

LINUX DO 主题《如果你愿意，我不愿意啊 GPT5.4》集中吐槽的不是某个单句，而是整套尾句模板：

- 如果你愿意
- 如果你需要
- 你要是愿意
- 你只要告诉我
- 我稳稳地接住
- 要不要我顺手

来源：[如果你愿意，我不愿意啊 GPT5.4](https://linux.do/t/topic/1699921)

### 3. 2026-03-28: Claude 也被认为“学坏了”

LINUX DO 主题《坏了 claude跟gpt学坏了》里，用户直说“不想要被稳稳地接住”，评论区把 `接住 -> 收口 -> 落盘` 当成一整套黑话链条。

来源：[坏了 claude跟gpt学坏了](https://linux.do/t/topic/1841752)

### 4. 2026-04-07: GPT-5.4 口癖集中爆发

LINUX DO 主题《受不了gpt5.4了，"我不瞎猜，如果你愿意"》里，用户和评论区集中列出一串高频模式：

- `我先 XXX，再顺手 XXX。如果你愿意，我可以 XXX`
- `不是 X，而是 Y`
- `如果你要，我下一步可以直接……`
- `稳稳地接住你`
- `一句话总结`
- `不绕`
- `抓手 / 收口 / 回归 / 验证`

该串讽刺性复刻获得高赞，说明这些短语已经从“偶发体感”变成“群体可识别模板”。

来源：[受不了gpt5.4了，"我不瞎猜，如果你愿意"](https://linux.do/t/topic/1916263)

### 5. 2026-04-10: Claude 与 GPT 被一起吐槽

LINUX DO 主题《claude 和 gpt 学坏了》里，评论直接说“被 GPT 狠狠污染了，让我稳稳接住你”。这说明用户感知到的不是单模型问题，而是跨模型风格收敛。

来源：[claude 和 gpt 学坏了](https://linux.do/t/topic/1940813)

### 6. 2025-10-18: 早期 Claude 口癖样本

在《发现一个Claude的口癖》里，用户提到 Claude 特别爱说“优雅”，评论区继续补充：

- `Perfect!`
- `You are absolutely right`
- `你完全正确`

这类模式和后来的中文“托举用户”属于同一路风格问题。

来源：[发现一个Claude的口癖](https://linux.do/t/topic/1059098)

### 7. 2026-04: GPT-5.x 中文口癖分类收集

LINUX DO 主题《gpt-5.x (codex) 中文口癖收集》对 GPT-5.x 系列的中文口癖做了系统分类：

- **暴力倾向类**：切、伤、砍一刀、补一刀、更狠、狠一点、狠狠干、打坏、拍板、拍脑门
- **废话连篇类**：好，行，我先、说穿、不踩坑、简单的说、不是 而是、我先 再、一句话总结
- **庸医问诊类**：痛点、根因、抠出来、我不猜、不靠猜、不瞎猜
- **不说人话类**：兜底、落盘、闭环、说穿、能吃、这轮、口径、拆开、说人话就是
- **单音节词类**：大量使用单字动词营造"干脆利落"的假象

这是目前最全面的分类收集之一，多个类别（暴力隐喻、装不猜、宣布说人话）此前未被系统覆盖。

来源：LINUX DO 站内帖，标题《gpt-5.x (codex) 中文口癖收集》。具体 URL 待补充。

## 二手整理

2026-04-08，Toy's Tech Notes 转述了一份社区整理结果，把中文 GPT 语癖概括为“机翻感、废话文学、互联网黑话、客服腔”，并强调负面清单和硬约束提示词的价值。

来源：[告别“AI味”：社区整理GPT模型语癖负面清单与提示词优化指南](https://www.80aj.com/2026/04/08/gpt-ai-language-optimization/)

## 观察结论

从这些案例里可以抽出四个稳定结论：

1. 高频问题不是单个词，而是成套模板一起出现。
2. 用户最反感的不是“简短”或“礼貌”，而是低信息量的表演性温柔。
3. `GPT-5.4` 的主要问题更像“邀约尾句 + 模板判断 + 假安抚”。
4. `Claude` 在中文里更早暴露的是“过度认同用户”和某些审美口头禅，之后也出现了被认为向 GPT 风格收敛的样本。

## 使用边界

LINUX DO 页面明确写了禁止 AI 代写站内内容。因此这份案例库只能用于研究和风格治理，不能拿本 skill 去替用户生成要发布到该站的帖子或评论。
